React Sorcerer

View project on GitHub

Sorcerer is a react component that builds a srcset for your img tags. Source sets make sure that you always have the right size image for the window size, increase site performance, and save your users bandwidth.

  • Open your network tab
  • Shrink your window all the way down to one side
  • Hit command-shift-r
  • Drag your window back to full size

You will see that when your window was small, the browser only downloaded a small image file. As your window became bigger, the browser downloaded the corresponding larger files.
